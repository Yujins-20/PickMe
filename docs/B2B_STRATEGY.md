# Phase 3: B2B 수익화 전략

## 1. 타겟 고객

### A. 중소기업 (10-50명)
**문제점**:
- 팀 점심/회식 결정에 매번 시간 낭비
- 복지 혜택 우선순위 결정 어려움
- 사내 이벤트 기획 시 의견 수렴 비효율

**솔루션: PickMe for Teams**
- 월 ₩49,000 (10명 포함)
- 팀 워크스페이스
- 투표 히스토리 관리
- 관리자 대시보드

**예상 객단가**: ₩49,000 × 100개 기업 = ₩4,900,000/월

---

### B. 대기업 (100명+)
**문제점**:
- 전사 설문조사 비용 높음 (외주 수백만원)
- 직원 복지 우선순위 파악 어려움
- 부서별 의견 차이 조율 필요

**솔루션: PickMe Enterprise**
- 월 ₩500,000+ (맞춤 견적)
- SSO 로그인 (SAML)
- 부서별 워크스페이스
- API 연동
- 전담 계정 매니저

**예상 객단가**: ₩500,000 × 10개 기업 = ₩5,000,000/월

---

### C. 외식업체/프랜차이즈
**문제점**:
- 신메뉴 테스트 비용 높음
- 메뉴 구성 최적화 어려움
- 고객 선호도 데이터 부족

**솔루션: PickMe Menu Lab**
- 메뉴 A/B 테스트 플랫폼
- 고객 선호도 분석
- 경쟁사 메뉴 비교

**가격**: 프로젝트당 ₩1,000,000 ~ ₩5,000,000

---

## 3. 영업 전략

### A. 인바운드 마케팅
- **블로그 포스팅**: "팀 점심 결정, 이제 30초면 끝"
- **사례 연구**: "스타트업 X가 PickMe로 매주 2시간 절약"
- **무료 체험**: 14일 전액 환불 보장

### B. 아웃바운드 영업
**타겟 리스트**:
- 네이버/카카오/쿠팡 등 IT 기업
- 배달의민족/요기요 (외식업 데이터 활용)
- 대기업 HR 담당자

**영업 스크립트**:
```
"안녕하세요, CIKM 학회 논문 기반 의사결정 플랫폼 PickMe입니다.
귀사의 팀원들이 매주 점심/회식 결정에 얼마나 시간을 쓰고 계신가요?
PickMe는 AI 알고리즘으로 3-4번의 클릭만으로 팀 전체의 선호도를 
정확하게 파악합니다. 14일 무료 체험 진행해보시겠습니까?"
```

### C. 파트너십
1. **HR SaaS 연동**: Flex, 자란다 등
2. **복지몰 제휴**: 티켓몬스터, 위메프오
3. **워크스페이스 툴**: Slack, Notion 연동
