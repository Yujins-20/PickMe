# Phase 4: 데이터 수익화 전략

## 🎯 핵심 아이디어

PickMe가 수집하는 **페어와이즈 비교 데이터**는 평점보다 훨씬 정교한 선호도 정보입니다.

### 왜 가치있는가?
```
기존 평점 데이터:
- "이 맛집 4.5점" (절대 평가)
- 맥락 없음
- 조작 가능

PickMe 데이터:
- "사용자들이 A보다 B를 70% 선호" (상대 평가)
- 실시간 선택의 순간
- 조작 어려움
- 인구통계학적 세그먼트 가능
```

---

## 1. 음식점/프랜차이즈 인사이트

### A. 맛집 프로모션 플랫폼

**제품**: "PickMe Insights"
- 우리 동네 20대 여성이 선호하는 메뉴 TOP 10
- 경쟁사 대비 우리 가게의 선호도
- 시간대별/날씨별 선호 메뉴 변화

**타겟 고객**:
- 개인 음식점 사장님
- 프랜차이즈 본사 (BBQ, 굽네치킨 등)
- 외식 컨설팅 업체

**가격 모델**:
```
기본: ₩99,000/월
- 내 가게 데이터만 열람
- 월간 리포트

프리미엄: ₩299,000/월
- 경쟁사 비교 분석
- 실시간 대시보드
- 메뉴 추천 AI

프랜차이즈: ₩5,000,000/월
- 전국 지점 데이터
- 신메뉴 A/B 테스트
- 전담 데이터 분석가
```

---

### B. 신메뉴 테스트 플랫폼

**사용 사례: 맥도날드 신메뉴 출시 전 테스트**

```typescript
// 1. 타겟 유저 모집
const testGroup = {
  location: '서울 강남',
  ageRange: [20, 35],
  preferences: ['버거', '패스트푸드'],
  size: 1000,
};

// 2. 테스트 실행
const menuTest = {
  candidates: [
    '불고기 버거',
    '갈릭 슈림프 버거', 
    '트러플 머쉬룸 버거',
  ],
  existingMenu: ['빅맥', '상하이 버거'],
  duration: '2주',
};

// 3. 결과 분석
const results = {
  winner: '갈릭 슈림프 버거',
  confidence: 87%,
  insights: [
    '20대 여성에게 가장 인기',
    '빅맥보다 65% 더 선호',
    '예상 가격: 7,500원 적정',
  ],
};
```

**가격**: 프로젝트당 ₩3,000,000 ~ ₩10,000,000

## 2. 유통/이커머스 인사이트

### A. 제품 선호도 분석

**타겟**: 쿠팡, 마켓컬리, 무신사 등

**제공 데이터**:
- 유사 제품 간 선호도 비교
- 가격 민감도 분석
- 패키징/디자인 선호도

**활용 사례**:
```
"여성 고객들이 A 디자인보다 B 디자인을 2:1로 선호
→ 패키징 변경 의사결정 근거"

"이 가격대에서는 품질보다 가격이 더 중요한 요소
→ 프로모션 전략 수립"
```

**가격**: API 호출당 과금 또는 월 구독

## 3. 광고 플랫폼 데이터

### A. 네이티브 광고

**맛집 투표에 자연스럽게 삽입**:
```
사용자가 "강남 돈까스" 투표 생성
→ 후보 10개 중 2-3개는 스폰서 맛집
→ 사용자는 자연스럽게 비교하며 선택
→ 맛집은 진짜 선호도 데이터 획득
```

**장점**:
- 사용자 경험 해치지 않음 (이미 비교 과정의 일부)
- 맛집은 정확한 타겟팅 (이미 해당 음식 찾는 사람)
- 높은 전환율

**가격**:
- CPM: ₩10,000 (노출 1,000회당)
- CPC: ₩500 (클릭당)
- CPA: ₩5,000 (방문/예약당)

---

### B. 선호도 기반 타겟팅

**예시: 치킨 브랜드 광고**
```
타겟팅 조건:
- 치킨을 자주 선택하는 사용자
- 프리미엄 가격에도 품질 선호
- 20-30대 남성

→ BBQ 프리미엄 치킨 광고 노출
→ 전환율 일반 광고 대비 5배 ↑
```

## 4. 익명화된 빅데이터 판매

### A. 트렌드 리포트

**제품**: "PickMe Food Trends Report"
- 월간/분기 외식 트렌드 리포트
- 세대별/지역별 선호도 변화
- 신흥 메뉴/트렌드 조기 감지

**타겟 고객**:
- 투자사 (외식 업계 투자 판단)
- 시장조사 기관
- 언론사 (트렌드 기사 작성)

**가격**: 
- 월간 리포트: ₩500,000
- 맞춤 분석: ₩5,000,000+

---

### B. API 판매

**제공 엔드포인트**:
```typescript
// 1. 지역별 인기 음식
GET /api/insights/popular-foods?location=강남&age=20-30

// 2. 메뉴 간 선호도 비교
GET /api/insights/preference?itemA=짜장면&itemB=짬뽕

// 3. 트렌드 점수
GET /api/insights/trend-score?keyword=마라탕&period=3months
```

**가격 모델**:
- Starter: ₩99,000/월 (1,000 calls)
- Business: ₩490,000/월 (10,000 calls)
- Enterprise: Custom (무제한)

## 5. 수익 시뮬레이션 (3년차)

### 맛집 인사이트
```
기본 플랜: 200개 × ₩99,000 = ₩19,800,000/월
프리미엄: 50개 × ₩299,000 = ₩14,950,000/월
프랜차이즈: 5개 × ₩5,000,000 = ₩25,000,000/월
신메뉴 테스트: 연 10건 × ₩5,000,000 = ₩50,000,000/년
```

### 광고 수익
```
네이티브 광고:
- 월 100만 노출 × ₩10 CPM = ₩10,000,000/월

선호도 타겟팅:
- 월 5만 클릭 × ₩500 CPC = ₩25,000,000/월
```

### 데이터 판매
```
트렌드 리포트: 30개 기업 × ₩500,000 = ₩15,000,000/월
API 구독: 100개 × ₩99,000 = ₩9,900,000/월
```
