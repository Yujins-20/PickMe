# PickMe 최종 실행 체크리스트 ✅

## 🚨 에러 해결 (지금 당장!)

### Mac에서 "next: command not found" 해결
```bash
# 1단계: 프로젝트 폴더로 이동
cd pickme-app

# 2단계: 자동 설정 스크립트 실행
chmod +x setup.sh
./setup.sh

# 또는 수동으로:
rm -rf node_modules package-lock.json .next
npm cache clean --force
npm install
npm run dev
```

**여전히 안 되면:**
```bash
# npx 사용
npx next dev
```

## 🔥 위험 요소 & 대응

### Risk 1: 초기 유저 확보 실패
**징후**: 2주 후에도 100명 미만
**대응**:
- 더 공격적 홍보 (Reddit, 해외 커뮤니티)
- 친구 추천 이벤트 (추천인에게 Pro 1개월)
- 인플루언서 협찬

### Risk 2: 유료 전환 안 됨
**징후**: 6개월 후에도 전환율 0.5% 미만
**대응**:
- 무료 플랜 제한 강화
- 프리미엄 기능 추가 (AI 분석, 히스토리)
- 7일 무료 체험

### Risk 3: 경쟁사 출현
**징후**: 유사 서비스 런칭
**대응**:
- 알고리즘 우위 강조 (논문 기반)
- 빠른 기능 개발
- 커뮤니티 구축

### Risk 4: 번아웃
**징후**: 개발 속도 저하, 동기 상실
**대응**:
- 주 1일 완전 휴식
- 멘토/동료와 정기 미팅
- 작은 성과 축하
